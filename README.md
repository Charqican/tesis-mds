### uv installation

This project was build using the uv package manager. uv its an extremely fast and simple Python package and project manager, written in Rust.
- on windows
```bash
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

- on macOs & Linux

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```
More details here [uv installation page](https://docs.astral.sh/uv/getting-started/installation/)
